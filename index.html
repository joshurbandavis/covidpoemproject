How to Contribute

the night air COVID poetry project is a community poem created using techniques similar to the french surrealist poets. One of these surrealist parlor games, called "Exquisite Corpse" was intended to democratize poetry creation by turning creative writing into a social game.

Here, we invite you to collaborate in a massive surrealist poetry game. To contribute to the collaborative poem, answer all of the questions below using one or two words.
